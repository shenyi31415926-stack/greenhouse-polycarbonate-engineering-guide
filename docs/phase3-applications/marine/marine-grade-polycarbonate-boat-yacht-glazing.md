# Marine-Grade PC Glazing for Boat and Yacht Builders

| Requirement | Spec | Standard |
|------------|------|----------|
| UV | 10yr+ in salt | ASTM G154 marine cycle |
| Impact | Wave slam, debris | ISO 12216 |
| Hard coat | Salt-spray resistant | ASTM B117 (500h+) |
| Fire | IMO FTP Code (SOLAS) | IMO A.754 |

Marine-grade uses polysiloxane with salt-spray additive. Water reflection amplifies UV 1.8-2.5x vs land. Edge seal critical: capillary action draws salt water into unsealed edges → delamination.
