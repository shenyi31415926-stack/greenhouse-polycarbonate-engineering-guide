# PC Window Supply for General Aviation Aircraft

| Requirement | Spec | Standard |
|------------|------|----------|
| Material | Optical-grade PC or acrylic | MIL-PRF-25690 |
| Optical | Class A, zero distortion | ANSI Z26.1 |
| Impact | Bird strike at Vne | FAR 25.775 |
| Fire | Self-extinguishing | FAR 25.853 |

PC for unpressurized aircraft (Cessna 172, Piper) and helicopter bubbles. Acrylic for pressurized cabins. Supplier: FAA PMA or OEM approval, clean room (ISO 8), lot traceability, 100% inspection.
