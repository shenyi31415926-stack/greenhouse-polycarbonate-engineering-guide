# Polycarbonate Security Glazing for Guard Booths and Checkpoints

Security glazing for fixed installations — guard booths, checkpoints, embassy windows, bank teller screens — requires ballistic protection plus everyday durability.

## Requirements for Fixed Security Glazing

| Requirement | Spec |
|------------|------|
| Ballistic | NIJ II to IIIA (threat-dependent) |
| Forced entry | 5+ minute delay (ASTM F1233) |
| UV stability | Outdoor, 10yr+ no yellowing |
| Anti-glare | Night shift visibility |
| Scratch resistance | Daily cleaning, weather |
| Thickness | 18-25mm (NIJ IIIA equivalent) |

## Level 1 vs Level 2 Security

| Level | Threat | PC Thickness | Cost/m² | Typical Use |
|-------|--------|-------------|---------|-------------|
| Level 1 (low) | Smash-and-grab, hand tools | 8-12mm | $200-400 | Retail, office |
| Level 2 (med) | Ballistic handgun (NIJ II-IIIA) | 18-25mm | $800-1,500 | Bank, guard booth |
| Level 3 (high) | Rifle (NIJ III+) | 40-60mm glass/PC | $2,500-5,000 | Embassy, military |

## Q: Polycarbonate alone or glass-clad polycarbonate?

For Level 1-2 (handgun): monolithic PC with hard coat is sufficient and lighter. For Level 3 (rifle): glass-clad PC laminate is required — glass absorbs the initial impact, PC catches spall and provides the final barrier.
