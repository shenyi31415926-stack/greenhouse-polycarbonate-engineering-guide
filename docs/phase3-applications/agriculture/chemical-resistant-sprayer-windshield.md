# Chemical-Resistant PC for Sprayer Vehicle Windshields

| Chemical | Test |
|---------|------|
| Glyphosate | 24h immersion, no etching |
| Diesel | 1h contact, no degradation |
| Alkaline cleaner (pH 12) | 100 cycles, no hazing |

Require ISO 2812 data from supplier. Self-propelled sprayer OEMs need: curved forming, exterior hard coat, interior anti-fog, FEA for road travel at 60 km/h.
