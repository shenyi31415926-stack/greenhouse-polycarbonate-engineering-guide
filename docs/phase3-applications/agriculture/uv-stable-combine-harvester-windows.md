# UV-Stable PC for Combine Harvester Windows

Combines operate 12-14 hrs/day in peak UV. Requirements: two-sided co-extrusion UV (interior reflection degrades single-sided), polysiloxane hard coat (grain dust abrasive), 8-10mm thickness (vibration damping), edge-sealed (dust in flutes causes fogging).

Check warranty exclusions for "agricultural use" and "abrasive environments."
