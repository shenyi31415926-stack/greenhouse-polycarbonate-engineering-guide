# Tinted vs Clear PC for Recreational Vehicle Windshields

| Tint | Transmission | Heat Rejection | Best For |
|------|-------------|---------------|----------|
| Clear | 88-90% | None | Fleet, night use |
| Light smoke | 50-60% | Moderate | Personal carts |
| Dark smoke | 25-35% | High | Desert/sunbelt |
| Bronze | 40-50% | High | Premium |

Always co-extruded UV regardless of tint. Dark tint without UV coating still transmits UV-A. Check state regulations for street-legal UTV tint limits.
