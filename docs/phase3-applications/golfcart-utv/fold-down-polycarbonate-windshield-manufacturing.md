# Fold-Down PC Windshield Manufacturing for Utility Vehicles

| Component | Material | Process |
|-----------|---------|---------|
| Panels | 4mm PC + hard coat | Die-cut |
| Hinge | Aluminum/SS | Riveted |
| Edge trim | PVC U-channel | Press-fit |

Tolerances: dimensions ±0.5mm, hole position ±0.25mm. Hard coat before die-cutting. Hinge alignment critical.
