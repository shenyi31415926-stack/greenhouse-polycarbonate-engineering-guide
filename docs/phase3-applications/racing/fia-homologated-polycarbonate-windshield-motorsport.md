# FIA-Homologated Polycarbonate Windshield Supply for Motorsport

Motorsport windshields are the only polycarbonate application where failure mode is tested at 300+ km/h with 3.5g lateral loads. FIA homologation is non-negotiable.

## FIA Windshield Requirements

| Requirement | Spec | Standard |
|------------|------|----------|
| Material | Polycarbonate, minimum 4mm | FIA Article 275 |
| Optical | Class A, no distortion | FIA Appendix J |
| Hard coat | Exterior abrasion resistant | FIA approved list |
| Marking | FIA homologation number etched | Mandatory |
| Flame | Self-extinguishing | FIA Std 8861 |

## FIA Homologation Process

1. Manufacturer submits material sample + construction details to FIA
2. FIA-authorized lab tests: optical, impact, flame, abrasion
3. If passed: FIA issues homologation number
4. Every production windshield must be etched with this number
5. Annual re-certification audit

Only FIA-homologated windshields can be used in FIA-sanctioned competition. Using non-homologated PC is an immediate scrutineering failure.
