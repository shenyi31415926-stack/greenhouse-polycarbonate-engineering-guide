# Polycarbonate Side and Rear Window Supply for Rally and Off-Road Racing

Rally car windows face gravel impact at 150+ km/h, mud abrasion, and rapid temperature cycling between stage and service. They are replaced more frequently than any other glazing application.

## Rally Window Requirements

| Component | Spec |
|-----------|------|
| Side windows | 4mm PC, hard coated both sides, sliding option |
| Rear window | 4mm PC, defrost element optional |
| Quarter windows | 3mm PC, fixed |
| Weight target | < 50% of glass equivalent |
| Attachment | Bolt-in or quick-release (rivnut) |

## Why Rally Teams Use Polycarbonate

Glass side windows shatter on first gravel stage. A WRC team replaces 2-4 glass windows per rally at €200-400 each plus service time. PC windows last full rallies — the cost savings across a 13-round WRC season are €15,000-30,000 in parts alone.

## Scratch Management

Unlike circuit racing, rally windows accumulate surface scratches from continuous gravel and mud impact. Teams budget 2 sets of PC windows per season: a fresh set for tarmac rounds, a used set for gravel rounds where optical perfection matters less. Removable protective films (tear-offs) are standard for the windshield but less common for side/rear.
